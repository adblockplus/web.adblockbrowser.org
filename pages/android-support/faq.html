title=FAQs
{{many-permissions-question[question heading] Why does Adblock Browser for Android require so many permissions?}}

{{permissions-p-1[answer paragraph] Generally, browsers require more permissions than regular apps to be able to provide web applications the ability to use hardware functionality such as camera, microphone, etc. (e.g. uploading images from your camera).}}

{{permissions-p-2[answer paragraph] Since Adblock Browser is based on the Firefox platform, it asks for the exact same permissions to do the same things Firefox does. Android requires that you accept all permissions in order to install the app. By doing so, you recognize that you're aware of what Adblock Browser for Android is able to do.}}

{{permissions-p-3[answer paragraph] Adblock Browser for Android requests access to the following permissions:}}

{{permissions-title-1[question heading] Device & app history}}
{{permissions-detail-1[answer paragraph] Allows the app to view information about activity on the device, which apps are running, browsing history and/or bookmarks, e.g. importing bookmarks and browsing history from another browser.}}
{{permissions-title-2[question heading] Identity}}
{{premissions-detail-2[answer paragraph] May be used to find accounts on your device, read your contact information or modify accounts, e.g. syncing information, such as bookmarks, passwords and open tabs, across devices.}}
{{permissions-title-3[question heading] Location}}
{{permissions-detail-3[answer paragraph] Uses the device's location when you choose to share your location with a website like Google Maps.}}
{{permissions-title-4[question heading] Photos/Media/Files}}
{{permissions-detail-4[answer paragraph] May be used to read or modify information on your device's external storage, e.g. accessing video stored on an SD card to upload to social media platforms like Facebook and Twitter.}}
{{permissions-title-5[question heading] Camera}}
{{premissions-detail-5[answer paragraph] Uses the device's camera(s), e.g. uploading images or videos to social media platforms like Facebook and Twitter.}}
{{permissions-title-6[question heading] Microphone}}
{{permissions-detail-6[answer paragraph] Uses the device's microphone(s), e.g. voice-to-text recognition in Google Voice Search and WebRTC (Web Real-Time Communication).}}
{{permissions-title-7[question heading] Wi-Fi connection information}}
{{permissions-detail-7[answer paragraph] Allows the app to view information about Wi-Fi networking, such as whether Wi-Fi is enabled and names of connected Wi-Fi devices.}}

{{permissions-p-4[paragraph] If you need more information, please refer to this FAQ from Mozilla.}}

{{flash-question[Flash video support question] Will Adblock Browser support flash videos?}}
{{flash-answer[Flash video support answer] Flash videos do work on Adblock Browser, but you must install the legacy flash player first. However, installing Flash for Android is not recommended as it’s insecure and may contribute to memory loss and slowness. If you still want to do this, refer to this article from Mozilla.}}
{{firefox-sync-question[Firefox synchronization question] Will you offer Firefox sync?}}
{{firefox-sync-answer[Firefox synchronization answer] Many of you have asked for this, and we agree this is a critical feature. That’s why we are working on enabling this in a future release.}}
{{chrome-bookmark-question[Chrome bookmark import question] How do I import bookmarks from Chrome?}}
{{chrome-bookmark-answer[Chrome bookmark import answer] If the default system browser is Chrome, you should be able to import bookmarks from Settings > Customize > Import from Android. If, however, the default browser is NOT Chrome, there's unfortunately nothing we can do.}}
{{google-sync-question[Google account synchronization question] Will you support Chrome / Google Account (passwords, bookmarks, etc.) syncing when Chrome is not the default browser?}}
{{google-sync-answer[Google account synchronization answer] Unfortunately, this is currently not a feature in Firefox for Android and as such, would be considered new browser functionality, which for now, we'll leave to the folks at Mozilla.}}
{{add-ons-question[Add-ons installation question] Can I install add-ons like LastPass and Ghostery, or other third-party services?}}
{{add-ons-answer[Add-ons installation answer] Having started as a browser add-on ourselves, we feel strongly about wanting to support add-ons, and hope to support this in the future. For now, if having add-ons is a deal breaker, we recommend using Firefox for Android with Adblock Plus.}}
{{custom-filter-question[Adding custom filters question] Can I add custom filters or block elements?}}
{{custom-filter-answer[Adding custom filters answer] We are looking into offering this and hope to add it in a future release.}}
{{version-question[Chrome version question] Will you offer a version of Adblock Browser based on Chrome?}}
{{version-answer[Chrome version answer] Our goal is to offer Chrome-like features without building a separate version based on Chrome.}}
